Renamed `size` prop values for the Avatar component. See the following table for the new prop mappings.

| Before                    | After       |
| ------------------------- | ----------- |
| `size="extraSmall"`       | `size="xs"` |
| `size="small"`            | `size="sm"` |
| `size="medium"`           | `size="md"` |
| `size="large"`            | `size="lg"` |
| `size="xl-experimental"`  | `size="xl"` |
| `size="2xl-experimental"` | `size="xl"` |
